**BookBuddyВаш личный библиотекарь в Telegram - BookBuddy!

Расширьте свои горизонты с BookBuddy! Этот переволюционный бот для Telegram предлагает не просто хранить ваши электронные книги, но и наслаждаться их чтением с помощью интуитивного интерфейса.

Ключевые особенности:

Ваша собственная библиотека: Удобно добавляйте или удаляйте книги. Начните с "Марсианских Хроник" и постепенно наращивайте свою коллекцию!
Чтение в движении: Просто отправьте книгу в чат в формате .txt, и она автоматически будет добавлена в вашу библиотеку.
Закладки: Выделите важные моменты во время чтения и легко возвращайтесь к ним позже.
Команды для управления ботом:

/start - активировать бота.
/help - получить справочную информацию о командах.
/continue - продолжить чтение с места, на котором остановились.
/books - просмотреть коллекцию ваших книг.
/bookmarks - просмотреть все ваши закладки на одном экране.

Технические детали:

Бот создан на Python 3.8 с использованием следующих библиотек:

aiogram 3.0.*
psycopg2 2.9.6
python-dotenv 1.0.0
requests 2.31.0
Данные хранятся в базе данных PostgreSQL.

Инструкции по установке:

Клонирование репозитория:
bash
Copy code
git clone [ссылка на репозиторий]
Настройка виртуального окружения и установка необходимых пакетов:
bash
Copy code
python3.8 -m venv env
source env/bin/activate
pip install -r requirements.txt
Запуск бота:
Copy code
python bot.py
Важное примечание: При каждом запуске бота база данных инициализируется заново, что приводит к удалению всех данных. Если вы хотите сохранять данные между сессиями, отключите функцию setup_db в файле bot.py.
