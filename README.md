**BookBuddy - Ваш персональный библиотекарь в Telegram**

Обогатите свою читательскую жизнь с помощью BookBuddy! Этот инновационный бот для Telegram позволяет не только хранить ваши любимые книги в электронном виде, но и наслаждаться их чтением в удобном интерфейсе.

**Основные функции:**
- **Личная библиотека:** Добавляйте или удаляйте книги с легкостью. Начните с "Марсианских Хроник" и расширяйте свою коллекцию!
- **Чтение на лету:** Отправьте книгу в чат в формате .txt, и она автоматически добавится в вашу библиотеку.
- **Закладки:** Отметьте интересующие вас места прямо во время чтения и возвращайтесь к ним в любое время.

**Команды для работы с ботом:**
- `/start` - Запустить бота.
- `/help` - Получить информацию по командам.
- `/continue` - Продолжить чтение с последнего места.
- `/books` - Просмотреть вашу библиотеку.
- `/bookmarks` - Все ваши закладки на одном экране.

**Техническая информация:**

Бот разработан на Python 3.8 и использует следующие библиотеки:
- aiogram 3.0.*
- psycopg2 2.9.6
- python-dotenv 1.0.0
- requests 2.31.0

Для хранения данных используется база данных PostgreSQL.

**Шаги для установки:**
1. **Клонирование репозитория:**
```bash
git clone [Ссылка на репозиторий]
```
2. **Настройка виртуального окружения и установка пакетов:**
```bash
python3.8 -m venv env
source env/bin/activate
pip install -r requirements.txt

python bot.py
```

**Важно:** При каждом запуске бота, база данных инициализируется заново, что приводит к удалению всех данных. Если вы хотите сохранять данные между сессиями, отключите функцию `setup_db` в `bot.py`.
